<h2>객체 : 같은 범위에 속하는것을 제어할때 정리하기 편하게함</h2>



	객체를 이용하면 서로 담당하는 영역을 구별하여 코드를 정리할 수 있다.

<h3>선언하는 방법 </h3>



		var 객체이름 = {

			“프로퍼티” : “값”,

			“프로퍼티2” : 값 ,

            func(){}

           // 마지막은 ‘ , ‘ 를 붙이지 않음

		}



<h3>객체의 프로퍼티 값 받아오는 방법</h3>



	객체이름.프로퍼티

<h3>객체에 프로퍼티 추가하는 방법 </h3>



    객체이름.propertyName = … ;

    OR

    객체이름[“객체 이름”] =” 내용 “ ; <= 객체이름에 띄어쓰기를 넣고싶을때



<h3>객체에 함수를 추가할때</h3>



    객체이름.함수이름 = function(매개변수) {



    }

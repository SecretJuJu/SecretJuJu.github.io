<h2> 뭔가 새로운 for문 </h2>

	다른점 / 추가된점 이 있다면 ?
	for문에서 간단하게 객체 혹은 배열의 길이만큼 반복문을 돌릴 수 있다.
		ex) var 배열이름 = [11,”2번째’”,”3번째”,4,5];
			for( a in 배열이름){
				console.log(a + “:” + 배열이름[a]);
			}
		== > 0 : 11 / 1 : 2번째 // 2 : “3번째” ……
